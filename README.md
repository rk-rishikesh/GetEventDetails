# Problem Statement

> Given a contract address, catch the details of all the transfer events from that contract using web3

# Steps for execution

### Clone the project
```bash
  git clone https://github.com/rk-rishikesh/Footprints
```
### Install node modules
```bash
  npm install
```
### Run
```
  node index.js
```
